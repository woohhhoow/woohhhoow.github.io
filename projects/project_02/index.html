A tactical urbanism intervention that reclaims vehicle-dominated intersections for pedestrians, creating a fluid, multi-functional civic space driven by micro-climate analysis.

Climate-Responsive Canopy

Using environmental simulation tools to map solar radiation and wind corridors, proposing a modular shading system that lowers surface temperatures by 4°C during summer.
